How to Apply CSS to HTML

To include an external CSS file, you can use the <link> element inside the <head> of your HTML file, like this:

<link rel="stylesheet" href="styles.css">

How to Use JavaScript in a Webpage

To link an external JavaScript file, you can use the <script> tag inside your HTML file, like this:

<script src="script.js"></script>

You can also write inline JavaScript within your HTML file like this:

<script>
    alert('Hello, world!');
    </script>
